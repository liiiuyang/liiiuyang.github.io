自动化测试

open 打开网址 Target
click 点击 Target(id=kw ) （定位要点击的ID）
type 输入内容 Target (id=kw ) Value (要输入的内容)
click Target（id=su）
refresh 刷新 没有参数
goBack 返回
pause 暂停 Target(5000)单位为毫秒
windowMaximize 窗口最大化
select模拟人工单击下拉列表框 value(里面填定位到的值) label(里面填真实的值)
